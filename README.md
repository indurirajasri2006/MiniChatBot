# 🤖 MiniChatBot

A simple AI-powered chatbot application built using **Python** and **Streamlit**. The chatbot provides an interactive interface where users can ask questions and receive AI-generated responses in real time.

---

## 📌 Project Overview

MiniChatBot is designed to demonstrate how conversational AI can be integrated into a web application. It provides a clean and user-friendly interface for interacting with an AI assistant.

---

## ✨ Features

- 💬 Interactive chat interface
- 🤖 AI-generated responses
- ⚡ Fast and responsive Streamlit web application
- 🖥️ Simple and clean user interface
- 🔄 Real-time conversation experience

---

## 🛠️ Technologies Used

- Python
- Streamlit
- OpenAI API
